### 원활한 iOS 개발을 위한 협업 규칙

- #### 커밋 규칙
> ##### 커밋 메세지는 자유로운 형식으로 작성하되 어떤 작업을 했는지, 무엇이 해결되었는지 상세히 작성해주세요!

- #### 이미지 네이밍 규칙
> ##### GOOUT_###Image 형식으로 네이밍해주세요! Ex) GOOUT_ProfileImage

- #### PR 규칙
> ##### 최소 두명 이상의 리뷰어를 추가하고 라벨을 추가해주세요!
